Below you can read more about some of the different types of tools that are usually used in photoshop, hopefully you will get an idea of what they are and how they can be used.

Select Tools

Select tools in Photoshop enable users to precisely choose and manipulate specific areas of an image. These tools include the Marquee, Lasso, Magic Wand, and Quick Selection tools, allowing for accurate selection.

Measuring Tools

Measuring tools provide users with the ability to measure distances, angles, and other dimensions within an image. These tools include the Ruler to help ensure accuracy and The eydropper to allows users to sample colors from images.

Cropping Tools

Cropping tools allow users to trim or resize the canvas or selected areas of an image. The Crop Tool enables precise cropping by adjusting the dimensions and aspect ratio, while the Perspective Crop Tool helps correct perspective distortion.

Navingation Tools

Navigation tools assist users in navigating and zooming within an image or document. The Hand Tool allows for panning across the canvas, while the Zoom Tool enables users to zoom in and out for detailed editing.

Painting Tools

Painting tools in Photoshop are used to create and manipulate images with brushes, colors, and textures. These are tools lik The Brush which allows users to paint using various brush and the History brush tool that allows users to selectively restore parts of an image to a previous state.

Typing Tools

Typing tools facilitate the addition and manipulation of text within an image or document. The Type Tool allows users to create text layers with customizable fonts, sizes, and styles, while the Vertical Type Tool enables vertical text orientation for specific design purposes.
